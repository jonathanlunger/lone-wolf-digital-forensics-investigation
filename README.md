# Lone Wolf Digital Forensics Investigation

## Overview

This repository contains a digital forensic case study involving the examination of a laptop hard drive image and memory dump for evidence of planning, preparation, and potentially violent intent.

The investigation correlated locally stored documents, browser activity, cloud synchronization records, deleted artifacts, and memory-resident communications to reconstruct the subject’s activity.

## Investigation Objectives

The examination focused on determining whether:

- Recovered activity demonstrated structured planning
- Browser research aligned with locally stored documents
- Cloud platforms were used to preserve or duplicate relevant files
- Memory analysis revealed communications not fully available on disk
- Timeline evidence showed a progression from research to preparation
- Artifacts could be attributed to a specific user account

## Tools Used

- AccessData Forensic Toolkit 8.2
- MemoryScope 0.5.0
- Antivirus scanning
- MD5 hash verification
